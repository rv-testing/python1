# Flask Example 1

## Setup

1. Create and activate a virtual environment:
   - Windows (PowerShell):
     python -m venv .venv
     .venv\Scripts\Activate.ps1

2. Install dependencies:
   pip install -r requirements.txt

3. Run the app:
   python app.py

## Test

- Open http://127.0.0.1:5000/
- Open http://127.0.0.1:5000/health
